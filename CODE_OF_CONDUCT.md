# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## In short

- Assume good faith. Extend the same to others.
- Disagreement with the rubric is welcome and expected. Personal attacks on contributors are not.
- Reports of unacceptable behaviour go to the maintainers listed in [`.github/CODEOWNERS`](.github/CODEOWNERS). Reports are handled confidentially.

## Scope

This code of conduct applies to all project spaces: this repository, associated issue trackers, discussion venues, and any events held under the project's name.
