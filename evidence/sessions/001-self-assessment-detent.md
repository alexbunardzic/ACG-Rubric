## Why this session exists

The rubric assesses the integrity of the practice by which code is produced with AI agents. If the authors of the rubric are exempt from it, the rubric is a marketing asset. If the authors are subject to it, and visibly fail it, and then fix it — the rubric is a standard.

This session is the launch. It is intentionally the first thing published, because the most persuasive sentence in the README should link to the authors' own failing grade.
